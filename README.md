# Prospect

The purpose of this code is to create a probability heatmap of Georgia predicting the likelihood that garnet occurs at a given location.

## Data sources & attribution

- **USGS Mineral Resources Data System (MRDS)** — U.S. Geological Survey, public domain. https://mrdata.usgs.gov/mrds/
- **Macrostrat** — University of Wisconsin–Madison, data licensed CC-BY 4.0. https://macrostrat.org
